# 📒 Contact Book (Python)

## **Project Description**

This repository contains two versions of a Contact Book application developed using Python. The project helps users manage contact information using lists and dictionaries.

## **Projects Included**

### **1. Basic Contact Book**
Features:
- Add Contact
- View Contacts
- Search Contact
- Delete Contact
- Update Contact

### **2. Menu-Driven Contact Book**
Features:
- Add Contact
- View Contacts
- Search Contact
- Delete Contact
- Update Contact
- Menu-Driven Interface

## **Concepts Used**

- Python
- Lists
- Dictionaries
- Loops
- If-Else Statements
- CRUD Operations

## **Files**

- `contact_book_basic.py` – Basic version of the Contact Book
- `contact_book_menu.py` – Menu-driven version of the Contact Book

## **Future Improvements**

- File Handling (Save contacts permanently)
- Input Validation
- Sorting Contacts
- GUI Version
